# Tumor Prediction using Decision Tree Classifier

## Project Overview
This project predicts whether a breast tumor is **Malignant** or **Benign** using the Decision Tree Classifier from Scikit-learn.

## Technologies Used
- Python
- Scikit-learn
- Jupyter Notebook

## Dataset
- Breast Cancer Wisconsin Dataset
- Loaded using `load_breast_cancer()` from Scikit-learn.

## Workflow
1. Import Libraries
2. Load Dataset
3. Prepare Features and Labels
4. Split Data into Training and Testing Sets
5. Train the Decision Tree Classifier
6. Predict Tumor Type
7. Evaluate Model Accuracy

## Features
- Decision Tree Classification
- Train-Test Split
- Tumor Prediction
- Model Accuracy Evaluation

## Result
The model predicts whether the tumor is **Malignant** or **Benign** and evaluates its performance using the test dataset.
